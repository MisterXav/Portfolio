# My Portfolio.

## Made with Vue, Vite and Tailswind

---

# Deploy

## Native

- Clone :
```sh
git clone https://github.com/MisterXav/Portfolio.git
```

- Install :
```sh
npm i
```

- Run :
```sh
npm run dev
```

## Docker (broken)

- Clone :
```sh
git clone https://github.com/MisterXav/Portfolio.git
```

- Run :
```sh
docker compose up -d
```
