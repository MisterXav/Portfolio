# Deploy

## Native

- Clone :
```sh
git clone <insert-link>
```

- Install :
```sh
npm i
```

- Run :
```sh
npm run dev
```

## Docker

- Clone :
```sh
git clone <insert-link>
```

- Run :
```sh
docker compose up -d
```
